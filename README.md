React16 + Redux + React-Router4 + MongoDB + Express Project

### Build Setup

```bash
# git clone
git clone https://github.com/laclys/React16-dev.git

# install dependencies
npm install

# Mongo Daemon
mongod --config /usr/local/etc/mongod.conf

# start Mongo
mongo

# run server
cd server
nodemon server.js

# serve with hot reload at localhost:3000
npm start

```
